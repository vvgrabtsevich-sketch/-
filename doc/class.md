# Class Diagram: Система "Золушка" (Управление временными ресурсами)

## Обзор

Диаграмма классов системы "Золушка" описывает иерархию персонажей, магические предметы и управление временными ресурсами.

---

## Иерархия классов

### Существа и персонажи

- **Существо** (абстрактный класс)
  - `сила: int`

- **Персонаж** (наследуется от Существо)
  - `состояние: String`
  - `танцевать()`

- **Золушка** (наследуется от Персонаж)
  - `потерятьТуфельку()`
  - `получитьМагическийПредмет(предмет: MagicItem)`

- **Принц** (наследуется от Персонаж)
  - `найтиТуфельку()`

---

### Управление ресурсами

- **Фея**
  - `выдатьМагическийПредмет(тип: String, времяИстечения: Date): MagicItem`

---

### Магические предметы

- **MagicItem** (абстрактный класс)
  - `прочность: int`
  - `времяИстечения: Date`
  - `разрушиться()`

- **Карета** (наследуется от MagicItem)
  - `прочность = 100`

- **Платье** (наследуется от MagicItem)
  - `прочность = 80`

- **Туфелька** (наследуется от MagicItem)
  - `прочность = 50`

---

### Управление временем

- **Таймер**
  - `текущееВремя: Date`
  - `проверитьПолночь(): Boolean`
  - `уведомитьОКонцеВремени()`

---

## Диаграмма классов

```plantuml
@startuml
!theme blueprint
skinparam classAttributeIconSize 0

' --- Иерархия существ ---
abstract class Существо {
    + сила: int
}

abstract class Персонаж extends Существо {
    + состояние: String
    + танцевать()
}

class Золушка extends Персонаж {
    + потерятьТуфельку()
    + получитьМагическийПредмет(предмет: MagicItem)
}

class Принц extends Персонаж {
    + найтиТуфельку()
}

' --- Управление ресурсами ---
class Фея {
    + выдатьМагическийПредмет(тип: String, времяИстечения: Date): MagicItem
}

' --- Магические предметы ---
abstract class MagicItem {
    + прочность: int
    + времяИстечения: Date
    + разрушиться()
}

class Карета extends MagicItem {
    + прочность = 100
}

class Платье extends MagicItem {
    + прочность = 80
}

class Туфелька extends MagicItem {
    + прочность = 50
}

' --- Управление временем ---
class Таймер {
    + текущееВремя: Date
    + проверитьПолночь(): Boolean
    + уведомитьОКонцеВремени()
}

' --- Связи ---
Фея ..> MagicItem : "<<create>>"
Золушка --> MagicItem : использует
Таймер --> Золушка : уведомляет о конце времени
Золушка --> Принц : взаимодействует

note right of Фея::выдатьМагическийПредмет
  **Управление ресурсами:**
  Фея выдает магические предметы
  с ограниченным временем действия.
end note

note bottom of Таймер
  Таймер отслеживает время
  и уведомляет Золушку о наступлении Полночи.
end note

@enduml
