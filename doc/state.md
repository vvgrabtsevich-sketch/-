# State Diagram: Жизненный цикл магических предметов (время действия до 00:00)

## Обзор

Диаграмма состояний описывает жизненный цикл магических предметов в системе "Золушка". Все магические предметы активны до наступления полуночи, после чего переходят в состояние "Истекший".

---

## Состояния
   Состояние      | Описание |
 |----------------|----------|
 | **Активный**   | Магический предмет используется Золушкой. |
 | **Истекший**   | Время действия магического предмета истекло (наступила полночь). |

---

## Диаграмма состояний

```plantuml
@startuml
!theme blueprint
skinparam state {
  BackgroundColor<<Expired>> #pink
}

title Жизненный цикл магических предметов (время действия до 00:00)

[*] --> Активный : <<create>>

state "Активный" as Активный
state "Истекший" as Истекший <<Expired>>
state c <<choice>>

' Все события идут в точку принятия решения
Активный --> c : проверитьВремя()

' Логика распределения потоков
c --> Активный : [время < 00:00] \n (продолжение использования)
c --> Истекший : [время >= 00:00] \n (наступила Полночь)

' Финал
Истекший --> [*] : <<destroy>>

note left of c
  **Точка выбора (Choice)**
  Проверяет условие:
  если время < 00:00,
  предмет остается активным.
end note

note right of Истекший
  Состояние изменилось,
  так как время действия
  истекло.
end note
@enduml
