# Sequence Diagram: Пошаговое взаимодействие в системе "Золушка"

## Обзор

Диаграмма последовательности описывает взаимодействие акторов (Фея, Золушка, Принц) и объектов (Карета, Платье, Туфелька, Таймер) в системе "Золушка".

---

## Участники
   Участник       | Описание |
 |----------------|----------|
 | **Фея**        | Создает и выдает магические предметы. |
 | **Золушка**    | Получает и использует магические предметы, танцует с Принцем. |
 | **Принц**      | Танцует с Золушкой, находит потерянную туфельку. |
 | **Карета**     | Магический предмет, используемый Золушкой. |
 | **Платье**     | Магический предмет, используемый Золушкой. |
 | **Туфелька**   | Магический предмет, который Золушка может потерять. |
 | **Таймер**     | Отслеживает время и уведомляет о наступлении полуночи. |

---
